比率ベースの美しいサイトを。

RatioKitはフレームワークなしでも真価を発揮します。 用意されたCSSクラスをHTMLタグに付与するだけで、複雑なレスポンシブレイアウトが完成します。

6:4 の黄金比レイアウト

flex64 クラスを指定するだけで、デスクトップでは完璧な比率で配置され、 モバイルでは自動的に最適化されたスタックレイアウトに切り替わります。

主要コンポーネント

HTML版は、クラス名のみで構築可能なシンプル設計です。

Responsive Cards

col3 などのクラスを親に指定するだけで、グリッドが構成されます。

Ratio Control

画像の比率を保ち、テキストとのバランスを自動調整します。

Pure CSS

JavaScriptなしで動作するため、あらゆる環境で利用可能です。

よくある質問

HTML版のメリットは？

軽量で依存関係がなく、どのようなCMSや静的サイトでもすぐに利用できる点です。

CSS変数のカスタマイズは？

:root 内の変数を上書きするだけで、全コンポーネントの色や余白を変更できます。
