# CMSコンポーネント実装パターン集

> **使い方**  
> スクリーンショットから「.card3」「.img_text」「.fl37」等のテンプレートクラスを選択し、大枠のレイアウトを構築。  
> 細かい装飾はTailwind CSSで調整する前提。

---

## 🎯 スクリーンショット → 実装フロー

```
1. レイアウト構造を特定（下記パターン表を参照）
2. テンプレートクラスで大枠を構築
3. 必要に応じて修飾子を追加
4. Tailwindで細かい装飾（色、余白、フォント等）
```

---

## 📦 主要コンポーネント一覧

### 1️⃣ カードレイアウト（`.cards*`）

**修飾子：**  `.img3-2`, `.imgCover`
| 見た目 | クラス | 備考 |
|--------|--------|------|
| 2列の等幅カード | `.col2` | |
| 3列の等幅カード | `.col3` | 最頻出 |
| 4〜6列 | `.col4` `.col5` `.col6` | |
| 可変幅（伸縮） | `.colflex` | 何個でも1行 |
| 固定幅 | `.colfix` | `--itenW`変数で幅指定 |


**基本HTML：**
```html
<div class="cards col3">
  <div class="">
    <img src="..." alt="...">
    <h3>タイトル</h3>
    <div><p>説明</p></div>
  </div>
  <!-- 繰り返し -->
</div>
```

---

### 2️⃣ 画像テキスト横並び（`.img_text*` ）

| 見た目 | クラス | 画像比率 |
|--------|--------|----------|
| 画像:テキスト = 3:7 | `.img_text .img30` | 30% |
| 画像:テキスト = 4:6 | `.img_text .img40` | 40% |
| テキスト:画像（逆） | `.img_text .is-rev` | 順序逆 |


**基本HTML：**
```html
<div class="img_text img30 ">
  <img src="..." alt="">
  <div>
    <h2>タイトル</h2>
    <p>説明文</p>
  </div>
</div>
```

---

### 3️⃣ panel（`panel*`）


**修飾子：** `.is_flow`
| 見た目 | クラス | 用途 |
|--------|--------|------|
| ステップ・フロー | `.is_flow` | 番号付き手順 |



**基本HTML（Q&A）：**
| Q&A形式 | `.fb_qa` | 質問と回答の繰り返し |

### 4️⃣ DLリスト（`.form_wrap .form_*`）

| 見た目 | クラス | 用途 |
|--------|--------|------|
| シンプル下線 | `.form_simple` | ニュース一覧 |
| 装飾なし | `.form_01` | 基本情報 |
| 背景付き | `.form_02` `.form_03` | 強調表示 |
| タイムライン | `.form_enkaku` | 沿革 |

**修飾子：** `.__scroll`（スクロール有効）

**基本HTML：**
```html
<div class="form_wrap form_simple">
  <dl>
    <dt>2025.01.15</dt>
    <dd>お知らせ内容</dd>
  </dl>
</div>
```

---

### 5️⃣ 比率指定Flexbox（`.fl*`）

| 見た目 | クラス | 比率 |
|--------|--------|------|
| 左右50:50 | `.flex55` | 均等 |
| 左30:右70 | `.flex37` | 3:7 |
| 左40:右60 | `.flex46` | 4:6 |
| 左60:右40 | `.flex64` | 6:4 |

**修飾子：** `.is_rev`（順序逆転）

**基本HTML：**
```html
<div class="fl37">
  <div class="H-bd">
    <h2>左側コンテンツ</h2>
  </div>
  <div>
    <img src="..." alt="">
  </div>
</div>
```

### 6️⃣ 見出し装飾（`.H-*`）

| 見た目 | クラス |
|--------|--------|

**基本HTML：**

### 7️⃣ ラッパー・背景制御

| 見た目 | クラス | 挙動 |
|--------|--------|------|
| 背景+余白 | `.wrapper` | padding付き |
| 画面幅背景 | `.wrapper100` | 全幅 |
| 画面幅背景+中身制約 | `.wrapper100in` | 背景のみ全幅 |
| 幅制約 | `.inBase` `.in1200` | 固定幅 |

## 🎨 クラスの3分類

### コンポーネントクラス
複数プロパティを一括設定（例：`.sheet`, `.card3`）

### アトミッククラス
単一プロパティ指定（例：`.AIC`, `.mt20`, `.TAC`）  
※ Tailwindで代替可能

### コンテナクラス（参考）
子要素を装飾（例：`.p-check`で中の`<p>`にチェックマーク）  
※ スクリーンショット再現ではTailwindで代替推奨
